Comandos principais de Git.

Clonar repositório:
git clone https://github.com/marianalm16/-Git-Flow---Testes.git
Ver status: git status
Adicionar arquivos: git add .
Criar commit: git commit -m "mensagem"
Enviar para o GitHub: git push
Atualizar projeto: git pull
Criar branch: git branch nome
Trocar de branch: git checkout nome
Criar e trocar de branch: git checkout -b nome
Deletar branch: git branch -d nome
Corrigir commit: git commit --amend
Reverter commit: git revert id
